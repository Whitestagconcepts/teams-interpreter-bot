Teams Interpreter Bot

Welcome to the Teams Interpreter Bot! This bot provides real-time translation and text-to-speech capabilities for your Microsoft Teams conversations.

Features

Message Translation

Send any message to the bot for instant translation between supported languages.

Example: Hello, how are you today?

Text-to-Speech

Convert text to natural-sounding speech in multiple languages.

Use the command: /speak Hello everyone

Meeting Interpretation

Add the bot to meetings for real-time interpretation.

(Coming soon)

Available Commands

  • /help - Display help information
  • /languages - Show supported languages
  • /translate <text> - Translate specific text
  • /speak <text> - Convert text to speech
  • /call <meeting-link> - Join a meeting (Coming soon)

Supported Languages

English (en-US)
Spanish (es-CO)
